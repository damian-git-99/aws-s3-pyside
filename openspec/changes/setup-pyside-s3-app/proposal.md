## Why

We need a desktop application to manage files in AWS S3 with a user-friendly GUI. Currently, there's no dedicated tool for this purpose in our ecosystem, requiring users to use the AWS CLI or web console.

## What Changes

- Set up a new PySide6 desktop application project structure
- Create main application window with basic UI framework
- Add project configuration files (pyproject.toml, requirements.txt)
- Implement minimal working window that opens successfully
- Set up project entry point and application initialization

## Capabilities

### New Capabilities
- `project-setup`: Basic PySide6 project structure and configuration
- `main-window`: Initial application window that opens and displays

### Modified Capabilities
- (none - new project)

## Impact

- New Python project with PySide6 dependency
- Desktop GUI application entry point
- No breaking changes (new project)
